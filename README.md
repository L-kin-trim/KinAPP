# KinApp - 游戏道具与战术管理应用

## 项目简介

KinApp 是一款专为游戏玩家设计的道具与战术管理应用，帮助玩家记录和管理游戏中的道具信息、投掷方法和战术策略。

## 功能特性

### 当前已实现功能

- **地图管理**：添加、删除不同的游戏地图，点击地图名字后，能查看到数据库里保存的道具投掷方法
- **道具管理**：为每个地图添加、编辑和删除道具信息
- **道具类型**：支持烟雾弹、燃烧瓶、手雷、闪光弹等多种道具类型
- **详细信息**：记录道具名称、投掷方法、位置等详细信息
- **图片支持**：为道具添加站姿、瞄准点、落地点的图片
- **分组显示**：道具信息按位置分组显示，便于查看
- **批量操作**：支持选择多个道具进行批量删除
- **战术管理**：添加、编辑、删除和查看游戏战术策略
- **战术类型**：支持RUSH、Split、Contain、Lurk、Default等多种战术类型
- **战术详情**：记录战术名称、描述、关联地图和类型等信息
- **战术分组**：战术按类型分组显示，支持展开/折叠查看
- **战术编辑**：支持修改现有战术信息
- **成员管理**：为战术添加成员信息，包括成员名称和角色/任务
- **数据存储**：使用SQLite数据库本地存储数据
- **界面优化**：
  - 战术主页面支持滑动，提升用户体验
  - 战术描述最多显示两行，超出部分用省略号表示
  - 优化了查看与修改按钮的大小和字体
  - 改进了删除战术时的UI布局

## 技术架构

- **开发语言**：Java
- **开发平台**：Android
- **UI组件**：Fragment、ViewPager、ListView、ScrollView等
- **数据存储**：SQLite数据库
- **图片处理**：Bitmap、MediaStore等

## 核心功能模块

### 1. 地图管理
- 添加新地图
- 删除现有地图
- 选择地图查看相关道具

### 2. 道具管理
- 添加新道具
- 编辑现有道具
- 删除道具
- 批量选择和删除道具

### 3. 道具信息录入
- 输入道具名称
- 选择道具类型（烟雾弹、燃烧瓶、手雷、闪光弹）
- 输入投掷方法
- 输入位置信息
- 添加站姿、瞄准点、落地点图片

### 4. 道具查看
- 按位置分组显示道具
- 点击道具查看详情
- 支持缩放查看道具图片

### 5. 战术管理
- 添加新战术
- 编辑现有战术
- 删除战术
- 查看战术详情
- 批量选择和删除战术
- 按类型分组显示战术
- 支持展开/折叠查看战术列表

### 6. 战术信息录入
- 输入战术名称
- 选择战术类型（RUSH、Split、Contain、Lurk、Default）
- 选择关联地图
- 输入战术描述
- 添加成员信息（成员名称和角色/任务）
- 支持修改现有战术信息

### 7. 战术查看
- 按类型分组显示战术
- 点击战术查看详情
- 显示战术名称、类型、地图、描述和成员信息

## 使用说明

### 添加地图
1. 打开应用，进入道具页面
2. 点击"添加地图"按钮
3. 在弹出的对话框中输入地图名称
4. 点击"确认"按钮

### 添加道具
1. 选择一个地图
2. 点击"添加道具"按钮
3. 填写道具信息：
   - 道具名称
   - 道具类型
   - 投掷方法
   - 位置信息
4. 点击"选择图片"按钮添加相关图片
5. 点击"保存"按钮

### 查看道具
1. 选择一个地图
2. 查看该地图下的道具列表
3. 按位置分组显示，点击道具可查看详情

### 删除道具
1. 长按道具进入选择模式
2. 选择要删除的道具
3. 点击"删除道具"按钮
4. 确认删除操作

### 删除地图
1. 选择要删除的地图
2. 点击"删除地图"按钮
3. 确认删除操作（注意：删除地图会同时删除该地图下的所有道具）

### 添加战术
1. 打开应用，进入战术页面
2. 点击"添加战术"按钮
3. 填写战术信息：
   - 战术名称
   - 战术类型（RUSH、Split、Contain、Lurk、Default）
   - 关联地图
   - 战术描述
   - 成员信息（成员名称和角色/任务）
4. 点击"保存"按钮

### 查看战术
1. 进入战术页面
2. 查看战术列表，按类型分组显示
3. 点击战术类型可展开/折叠查看该类型下的战术
4. 点击"查看"按钮查看战术详情

### 编辑战术
1. 进入战术页面
2. 找到要编辑的战术
3. 点击战术右侧的"修改"按钮
4. 修改战术信息，包括成员信息
5. 点击"保存"按钮

### 删除战术
1. 长按战术进入选择模式
2. 选择要删除的战术
3. 点击"删除战术"按钮
4. 确认删除操作

## 数据库结构

### 地图表 (map)
- id: 地图ID (主键)
- name: 地图名称
- icon_position: 图标位置

### 道具表 (daoju)
- id: 道具ID (主键)
- map_id: 关联的地图ID (外键)
- type: 道具类型 (1-烟雾弹, 2-燃烧瓶, 3-手雷, 4-闪光弹)
- position: 位置信息
- info_id: 关联的道具信息ID (外键)

### 道具信息表 (daoju_information)
- id: 信息ID (主键)
- throwing_method: 投掷方法
- stance_image_path: 站姿图片路径
- aim_point_image_path: 瞄准点图片路径
- landing_point_image_path: 落地点图片路径
- tool_name: 道具名称

### 战术信息表 (zhanshu_information)
- id: 战术ID (主键)
- map_id: 关联的地图ID (外键)
- type: 战术类型 (1-RUSH, 2-Split, 3-Contain, 4-Lurk, 5-Default)
- name: 战术名称
- description: 战术描述
- image_path: 图片路径
- video_path: 视频路径
- notes: 备注
- member1: 成员1名称
- member1_role: 成员1角色/任务
- member2: 成员2名称
- member2_role: 成员2角色/任务
- member3: 成员3名称
- member3_role: 成员3角色/任务
- member4: 成员4名称
- member4_role: 成员4角色/任务
- member5: 成员5名称
- member5_role: 成员5角色/任务

## 权限说明

- `READ_EXTERNAL_STORAGE`: 读取存储权限，用于选择图片
- `WRITE_EXTERNAL_STORAGE`: 写入存储权限，用于保存图片

## 修复的Bug

1. **战术主页面不可滑动**：修复了战术主页面无法滑动的问题，现在可以正常滑动查看所有战术
2. **修改界面未能继承已有内容**：修复了点击修改按钮后，修改界面未能显示保存在数据库中内容的问题
3. **保存后数据库未更新**：修复了修改战术信息后，点击保存按钮未能更新到数据库的问题
4. **数据库版本不一致**：修复了不同DAO类使用不同数据库版本导致的崩溃问题
5. **删除战术UI布局问题**：修复了点击删除战术时，确认删除按钮位置移动的问题
6. **战术描述显示问题**：修复了战术描述行数过多导致显示不美观的问题，现在最多显示两行，超出部分用省略号表示
